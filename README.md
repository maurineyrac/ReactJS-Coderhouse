## Para correr la aplicacion localmente seguir los siguientes pasos:
##### Abrir terminal desde Windows o Visual Studio Code 
1. Clonar repositorio 
```
git clone [repo-url]
```

2. Insalar dependencias
```
npm i
```
3. Correr app
```
npm run dev
```
